AMQP-SPY : guidebug.rb
======================

Simple GUI tool for show messages trafic and send (json) message to any routing_key.

![hmi](https://raw.githubusercontent.com/glurp/amqp-spy/master/hmi.png)

Tested on windows10/RabbitMQ.

Installation
============

> gem install bunny

> gem install Ruiby
